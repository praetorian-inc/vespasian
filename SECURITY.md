# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Vespasian, please report it responsibly.

**Email:** [security@praetorian.com](mailto:security@praetorian.com)

Please include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will acknowledge receipt within 48 hours and provide an initial assessment within 5 business days.

## Supported Versions

| Version | Supported |
|---|---|
| Latest release | Yes |
| Previous minor | Best effort |

## Responsible Use

Vespasian is a security tool built for authorized penetration testing and security assessments. Always obtain proper authorization before scanning any target. Coordinate with target owners and prefer staging environments during security assessments.
